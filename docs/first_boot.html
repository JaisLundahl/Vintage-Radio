First Boot & Hardware Calibration

This document details the sequential filesystem audit and the tactile "sweep" calibration for the tuning potentiometer.

1. Sequential Boot Audit

Upon detecting a "Virgin" SD card, the system performs a step-by-step audit to ensure the environment is ready for the SOA framework.

  • File Validation: The system checks for mandatory assets like /system/sound/wait.mp3.
  • Auto-Generation: If non-mandatory files (such as history.txt or local preference logs) are missing, the system generates them with default parameters.
  • MP3 Library Scan: If the device boots into MP3 Mode, it initiates a recursive scan of the 16 virtual bins to index the track count for each folder this can take a lot of time at first boot.

2. The Calibration Handshake

Calibration is a guided hardware-software handshake that maps the physical limits of the tuning potentiometer to the internal logic segments.

The Calibration Procedure

  1. Safety Interlock: Set the physical Volume knob to 0%. This prevents accidental triggers during normal playback.
  2. The First Limit: Turn the tuning knob to one outer edge (e.g., far left).
  3. Initiate: Push and hold the selector button for 5 seconds. The system records this as the MIN or MAX boundary.
  4. The Sweep: While continuing to hold the button, slowly rotate the tuner to the other outer edge (e.g., far right).
  5. Commit: Release the button at the second edge. The system captures this second value and calculates the total range.
Technical Note: The software uses these two captured 12-bit ADC values to mathematically divide the physical sweep into 16 equal segments (bins).

3. Persistence & Mapping

Once the button is released, the calibration state machine concludes and saves the data.

  • NVS Write: The calculated range is stored in the ESP32's Non-Volatile Storage, ensuring the radio stays calibrated after power-off.
  • Deadzone Calculation: The system automatically assigns "deadzones" between the 16 segments to trigger the wait.mp3 static effect during transitions.
  • EMA Reset: The Exponential Moving Average filters are re-primed with the new range to ensure immediate, smooth needle tracking.
