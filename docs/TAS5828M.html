TAS5828M Amplifier Driver Deep Dive

This module provides low-level I2C control for the Texas Instruments TAS5828M digital input Class-D amplifier.

1. Hardware Communication & Discovery

init()

Initializes the I2C interface and searches for the amplifier hardware.

  • I2C Configuration: Sets the clock speed to 100 kHz, which is the standard operational frequency for this amplifier.
  • Dynamic Volume Limits: Reads MASTER_REG_MAX from the SD card (/system/sound/max_volumes.txt) to set a hardware-level gain ceiling, preventing speaker damage.
  • Device Detection: Pings I2C address 0x60. If the device responds, s_ampFound is set to true.
  • Factory Reset: Sends a reset command (0x11) to TAS_REG_RESET_CTRL to clear previous DSP states and register values, followed by a 50ms delay for stabilization.

2. Volume and Mute Control

setVolumeAndMute(uint8_t volPct, bool mute, bool force)

Translates human-readable percentage (0-100) into 8-bit hardware register values.

  • Linear Mapping: Calculates the register value between a hardcoded minimum (0xC0) and the user-defined s_regMax.
  • State Optimization: Unless the force flag is true, the function returns early if the requested volume and mute state match the already applied values to reduce I2C bus traffic.
  • Device Control: Sets TAS_REG_DEVICE_CTRL to 0x0B for Mute/Deep Sleep or 0x03 for Playback.

3. Signal Path & Fault Recovery

ensurePlaybackPath()

Configures the internal routing of the amplifier and clears error states.

  • SAP Configuration: Sets the Serial Audio Port (SAP) control registers (1, 2, and 3) to ensure the I2S data format is correctly interpreted by the DSP.
  • Fault Clearing: Writes 0x80 to register 0x78. This is critical for recovering from "Clock Faults," which often occur when the ESP32 starts or stops the I2S bit clock.

4. Register Map Summary

The driver interacts with these primary registers on the TAS5828M:

Register Name Address Description
TAS_REG_RESET_CTRL 0x01 Resets DSP and digital registers.
TAS_REG_DEVICE_CTRL 0x03 Controls power state (Play/Mute/Sleep).
TAS_REG_VOLUME 0x4C Sets the digital volume gain.
TAS_REG_BOOK / PAGE 0x7F / 0x00 Memory mapping controls for multi-page register access.
