Main Application Orchestrator

The main.cpp file serves as the entry point and central coordinator, managing the system lifecycle, event dispatching, and power protection.

1. System Boot & Initialization

setup()

This function handles the sequential power-up of all hardware and software services.

  • Hardware Init: Starts Serial debugging, initializes the RGB LED via outputService(), and mounts the SD card using high-speed SPI.
  • Service Init: Boots InputService, AdcService, and AudioService.
  • Stable Mode Selection: Uses waitForStableSelector(2000) to ensure the user isn't actively turning the mode knob before deciding which mode to start.
  • Volume Priming: Sets the amplifier gain immediately to match the physical potentiometer before any audio begins playing.

2. Event Loop & Dispatcher

loop()

The main loop runs continuously, acting as the primary dispatcher for user input and system health checks.

  • Input Handling: Polls inputsGetEvent() and routes events (Rotary CW/CCW, Short/Long/Very Long Press) to the currently active mode (BT, MP3, or NET).
  • Mode Switching: If EV_MODE_CHANGED is detected, the system initiates a controlled Reboot via rebootFromMode(). This ensures a clean slate for the ESP32 when switching between complex tasks like Bluetooth and WiFi.
  • Volume Management: Updates the amplifier gain every 50ms to ensure smooth response while preventing I2C bus congestion.

3. Power and Calibration Features

Battery Protection

The loop monitors adcBattery_mV(). If the voltage drops below 12V (12000mV) for a sustained period, it triggers safeShutdown() to protect the battery from over-discharge.

Tuner Calibration

Implemented through EV_VERY_LONG_PRESS and RELEASE on the encoder button. This allows the user to set the physical Min/Max limits of the Tuner potentiometer, which are then persisted to the SD card.

4. SD Card File Structure

The system expects a specific directory structure on the SD card to function correctly:

/ (Root)
├── /folder1 ... /folder16 (MP3 Stations)
└── /system
    ├── /sound (wait.mp3, max_volumes.txt)
    ├── /netradio (WIFI.txt, stations.txt)
    └── /mp3 (mp3_folders.txt, mp3_state.txt)
